## `<span>` Рассчет молярной массы библиотеки

<span>

Дано:

* V=1,5 мкл
* 10000 разведении
* Среднее количесво ДНК = 1.014155 копий/мкл
* Средняя длина фрагмента = 678 bp
* Mbp(молярная масса одной пары) = 2 * 330 г/моль = 660 г/моль.

Решение:

Молярность библиотеки = $\dfrac{n библиотеки}{V библиотеки}$

M(молярная масса среднего фрагмента) = $660 * 678 = 447480$ г/моль

ncop (количество копий) = $1.014155 * 10^{-6} * 10000 = 1.014155 * 10^{-2}$ копий/л

n библиотеки = $\dfrac{m}{M} = \dfrac{N}{NA}$

$m = \dfrac{ncop * M}{NA} = \dfrac{1.014155 * 10^{-2} * 447480}{6.022 * 10^{23}} = 753,59 *10^{-23} $ г/л

Молярность библиотеки = $\dfrac{m}{M} = \dfrac{753,59 *10^{-23}}{447480} = 1.684 * 10^{-26} $ моль/л

</span>
